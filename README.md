# Synent Technologies Internship - Task 8
# House Price Prediction using Machine Learning

## Problem Statement

Develop a machine learning model to predict house prices using property-related features.

## Dataset Details

Dataset Name: House Price Prediction Dataset

Source: Kaggle

Target Variable:
- Price

Features:
- Bedrooms
- Bathrooms
- Sqft Living
- Floors
- Waterfront
- Condition
- Grade
- Year Built

## Approach

1. Loaded dataset.
2. Processed date information.
3. Performed feature engineering.
4. Selected numerical features.
5. Split dataset into training and testing sets.
6. Trained Random Forest Regressor.
7. Evaluated performance using RMSE and R² Score.
8. Created Streamlit application for predictions.
